# AI Knowledge Library

A personal knowledge repository focused on **Artificial Intelligence** and its **practical applications for companies** — from strategy and governance through to tooling, frameworks, and real-world use cases.

---

## Repository Structure

```
ai-knowledge-lib/
├── strategy/          # AI strategy, roadmaps, adoption frameworks
├── use-cases/         # Real-world AI use cases by industry & function
├── frameworks/        # Decision frameworks, evaluation criteria, maturity models
├── tools/             # AI tools, platforms, model comparisons
├── governance/        # AI governance, risk, ethics, and compliance
├── artefacts/         # Templates, checklists, canvases, and playbooks
├── research/          # Paper summaries, market reports, trend analyses
└── README.md
```

## Purpose

This repository serves as a living knowledge base to:

- Capture and organise insights on **enterprise AI adoption**
- Document **use cases** across industries and business functions
- Evaluate **AI tools, platforms, and models** for business readiness
- Develop **frameworks** for AI strategy, governance, and procurement
- Provide ready-to-use **artefacts** (templates, checklists, playbooks)
- Surface **market trends** and research findings

## How to Use

Each directory contains markdown files organized by topic. Start exploring:

1. **[Strategy](./strategy/)** — How to build and execute an AI strategy
2. **[Use Cases](./use-cases/)** — What AI can actually do for businesses
3. **[Frameworks](./frameworks/)** — Structured approaches to AI decisions
4. **[Tools](./tools/)** — What's available and how to pick the right ones
5. **[Governance](./governance/)** — Keeping AI safe, compliant, and ethical
6. **[Artefacts](./artefacts/)** — Practical templates to get started
7. **[Research](./research/)** — What the data and analysts are saying
